# Slimore Application

[Slimore](https://github.com/slimore/slimore) MVC single module application template.

## Install

`composer.json` :

```json
{
	"require" : {
		"slimore/slimore-application" : "*"
	}
}
```

composer install :

```shell
$ composer install
```
